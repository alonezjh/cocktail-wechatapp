# Cocktail鸡尾酒

> A wechatapp for cocktail

一款关于鸡尾酒的微信小程序，目前完成了5个一级界面。

数据使用easy-mock提供的接口。

鸡尾酒图片来源于Jo鸡尾酒APP。

后期将会完善该小程序，使用PHP搭建后台系统，并添加更多功能。
